# Thánh Gióng - The Legend (MVP)

Một tựa game 2D được phát triển bằng **Godot Engine**, lấy cảm hứng từ truyền thuyết hào hùng về Thánh Gióng của Việt Nam. 

Trong dự án này, người chơi sẽ theo chân một người anh hùng mang nón lá, khoác áo choàng đen, chiến đấu để bảo vệ dân làng khỏi các thế lực thù địch. Phiên bản hiện tại là MVP (Minimum Viable Product) tập trung vào các cơ chế gameplay cốt lõi, di chuyển và thiết kế nhân vật cơ bản.

## 🛠 Công nghệ sử dụng
* **Game Engine:** [Godot Engine](https://godotengine.org/) 
* **Ngôn ngữ lập trình chính:** GDScript (93.3%)
* **Khác:** Python, GDShader

## 📁 Cấu trúc thư mục
* `Assets/`: Chứa các tài nguyên đồ họa (hình ảnh, sprite nhân vật, nón lá, áo choàng...), âm thanh và font chữ.
* `Scenes/`: Chứa các scene của Godot (màn chơi, giao diện, nhân vật).
* `Scripts/`: Chứa toàn bộ mã nguồn GDScript điều khiển logic game.
* `tilesets/`: Chứa các bộ tile dùng để thiết kế bản đồ, cảnh quan trong game.
* `tools/`: Các công cụ và script hỗ trợ phụ trợ (Python/GDScript).

## 🚀 Hướng dẫn cài đặt và chạy thử
1. Tải và cài đặt **Godot Engine** (Khuyến nghị phiên bản tương thích với dự án).
2. Clone repository này về máy:
   ```bash
   git clone [https://github.com/hoaikhoitran/godot-ThanhGiong-mvp.git](https://github.com/hoaikhoitran/godot-ThanhGiong-mvp.git)
   ```
3. Mở Godot, chọn **Import** và trỏ tới file `project.godot` trong thư mục vừa tải về.
4. Nhấn nút **Play** (hoặc `F5`) để chạy game.
